## Description:

1. There are at least 6 domain specific languages in the system I made to write and publish this book, what are they?

## Definition:

Domain Specific Language (DSL): A language designed for a specific problem domain, not for general programming purposes.

## Answer:

Below are some of the DSLs used in the [crafting interpreters repository](https://github.com/munificent/craftinginterpreters)

1. .html

    creating web models/content.

2. .css & .scss

    styling html contents.

3. make

    build automation/build systems.

4. .htaccess

    configuring behaviours of apache http servers.

5. .md

    markup language for document formatting.

6. .gitignore

    define which files to be tracked by git.
